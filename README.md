# Human-Robot-Collaboration---Thesis

Effective Human-Robot Collaboration (HRC) requires understanding from the robot with respect to humans by generating dialog and/or physical action. This research investigates approaches for improving robot decision-making in a collaborative object-finding task. This thesis presents several different traditional supervised learning models in relation to fine-tuning large language models; for the prediction of robot intentions and corresponding actions in a collaborative object-finding task. The Light Gradient-Boosting Machine (LightGBM) model stands out as being on top of other classifiers with the most accurate response against the Generative Pre-trained Transformer (GPT) embedding system. Although Large Language Models (LLMs) are powerful language models, this research reveals their limitations in multimodal scenarios. However, some improvements were observed after data augmentation, which involved higher computational demands. These findings highlight the potential of traditional methods to develop smarter and context-aware systems for real-world applications.
